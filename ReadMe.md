#数据分析系统系统使用指南
##一.前提准备：
1).解压“资源文件”压缩文件到任意位置。  \
2).确定文件内含有mysql、idea、git、gradle、jdk的安装包。 \
3).网络查找以上几种插件或工具的安装方式，使用文件内的安装包进行安装。 \
4).linux 或者mac OX 用户自行网络下载安装对应软件。
